# node-tdd-template

## Getting Started


* Dependency Installation
```
npm ci
```

* Husky Installation
```
npx husky install 
```

* Husky Pre-Push Hook Installation
```
npx husky add .husky/pre-push "npm run lint && npm run test -- --passWithNoTests"  
```

* Starting the Program
```
npm start
```

* Running the Tests
```
npm run test
```
